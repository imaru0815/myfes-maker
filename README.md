# マイフェスメーカー Ver.2

公開用の全部入り版です。

## 機能
- フェス名入力
- フェスロゴ風デザイン
- テンプレート切替
- ステージ数 2〜4
- ステージ名変更
- ステージ色変更
- ステージごとの開始・終了時間変更
- アーティスト入力
- 自動タイムテーブル生成
- 各アーティスト枠クリックで開始時間・終了時間・名前を編集
- 各ステージ最後の出演者だけ90分
- 終演時間超え警告
- 「出演者数が多すぎて入りません」表示
- 画像保存
- X用1080×1350出力
- 共有

## Netlifyで公開
`index.html`, `style.css`, `app.js` をNetlifyにドラッグ＆ドロップ。

## Vercelで公開
GitHubにアップしてVercelでImport。
